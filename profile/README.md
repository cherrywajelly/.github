### 🗂️ Git Convention

| Header | 기능 | 
| --- | --- | 
| 🌈  Update | 변경 사항 업데이트 |
| 📍  feat | 새로운 기능 추가 |
| 🔨  fix | 버그 수정 | 
| 📝  docs | 문서 수정 | 
| 🎨  style | 코드 포멧팅 | 
| 🤖  refactor | 코드 리팩토링 | 
| ✅  test | 테스트 코드 | 
| 🚚  chore | 빌드 업무 수정, 패키지 매니저 수정 | 
| 💬  comment | 주석 추가 및 변경 | 
| ✂️  remove | 파일, 폴더 삭제 | 
| 🔧  rename | 파일, 폴더명 수정 | 



### 👻 Git Flow

| Header | 기능 | 
| --- | --- | 
| 📍  main | 최종 배포할 서비스 내용의 브랜치 |
| 🔨  dev | 주요 개발 브랜치, 이 브랜치를 기준으로 각자 작업한 기능을 merge | 
| 📝  feat | 기능 개발 브랜치, 개발이 완료 시 dev 브랜치에 merge | 
| 🎨  test | 테스트 코드 개발 브랜치, 개발 완료 시 dev 브랜치에 merge |
